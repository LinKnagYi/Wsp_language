## 安装WSP

```bash
vi /ect/profile
export WSPPATH=WSP所在目录
```
```bash
ln -s WSP所在目录/wsp /usr/bin
```
## 介绍

基于golang开发的解释型语言 使用wsp虚拟机，效率极高，当前版本 V2.0.0,有PHP的简单 Python的实用 Golang的效率

## 开发

```bash
wsp ./xxxx.wsp
```
